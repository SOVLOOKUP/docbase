# DocBase

基于文件系统的知识库

轻松构建，自动管理，极速搜索

## 特性

- 📂 支持多种文档格式：Markdown、TXT、PDF、Word、Excel、CSV、ZIP 等
- 🧩 插件化架构，易于扩展
- 🚀 实时文件监控与自动更新
- 🔍 强大的混合搜索（全文检索 + 模糊搜索 + 向量搜索）功能
- 🐳 支持 Docker 部署（plan）
