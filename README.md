## ✨ 特性

- 📂 支持多种文档格式：Markdown、TXT、PDF、Word、Excel 等
- 🧩 插件化架构，易于扩展，文档格式随心适配
- 🚀 实时监控文件变动，自动更新知识库数据
- 🔍 强大的混合搜索（全文检索 + 模糊搜索 + 向量搜索）功能
- 🐳 一键启动，支持 Docker Compose 部署
- ☁️ 支持 WebDAV 管理知识库
- 🔌 支持作为 Dify 外部知识库
- 💻 支持本地安装部署（Plan）
